# potential-octo-lamp

Word List Generator for Python

Transfer Text to Python List

Data Source: [1,000 most common US English words](https://gist.github.com/deekayen/4148741)

## Usage

    chmod +x main.py
    ./main.py --input wiki-100k.txt --output words.py
